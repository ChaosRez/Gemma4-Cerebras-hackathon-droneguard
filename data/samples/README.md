# Sample Data

Store demo scenarios here.

Recommended layout:

```text
data/samples/
|-- safe/
|   |-- frames/
|   |-- telemetry.csv
|   `-- mission.txt
`-- risky/
    |-- frames/
    |-- telemetry.csv
    `-- mission.txt
```

Do not commit sensitive flight logs or private location data. Synthetic telemetry is preferred for the hackathon demo.
